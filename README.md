# Attendance-Management-System
Attendance Management System adalah sistem manajemen absensi berbasis web yang dikembangkan menggunakan Laravel.  Sistem ini menyediakan fitur pengelolaan karyawan, pencatatan absensi, manajemen role &amp; user, laporan kehadiran, serta pengaturan profil perusahaan. Dirancang untuk mendukung kebutuhan operasional HR secara terstruktur dan efisien.
